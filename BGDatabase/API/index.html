API

This is basic BGDatabase API to use without CodeGeneration. Code Generation makes these API even easier to use and we strongly recommend to use it. More information about Code Generation is here

Following 4 classes are enough to know to work with BGDatabase

  1. BGRepo - Database itself
  2. BGMetaEntity - Database table
  3. BGEntity - Table row
  4. BGField - Table field
