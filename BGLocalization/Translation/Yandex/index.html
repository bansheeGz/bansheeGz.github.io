Yandex Translate

Setup

  1. Go to Configuration->Locale and add field of type string, named "yandexLanguage"
  2. Get a free API key for Yandex Translate as described here
  3. Go to Settings->Translation, paste your API key there and chose yandexLanguage field as "Language field"
  4. Go to Database->Localization and for each locale fill "yandexLanguage" field with the right language
  5. Chose any table with "localization" (Locale) type and string/text field type. Click on Setup and chose Yandex as translation service. Chose your source language and destination languages.
  6. Click "Run" to translate several rows. Click T↑ to translate one row to all locales. Click T↓ to translate current row to one locale
