What is it

Bezier curve editor for Unity 5.3

Features

  • Single Game Object (no additional objects for points)
  • A lot of settings
  • Ability to save settings on disk
  • Base math for Bezier curves

How to use it

Download and import BG Curve package from Unity asset store. Chose 'Game Object->Create Other->BG Curve' to create a curve. Then Ctrl+Shift+LeftClick to add a point at some distance from the camera or, alternatively, hover over an existing mesh and Ctrl+LeftClick to add a point snapped to a mesh. You can select several points and perform group operations. You can chose 'Settings' tab and make changes to a lot of settings parameters.
