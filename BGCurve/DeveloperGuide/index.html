Developer Guide

General information

You can add a curve, change it's position, rotation and scale, and also you can add, change and remove its points at runtime (Play mode).

Understanding Math

Math gives you access to some of the essential methods for dealing with splines. It calculates data by splitting every curve section to even parts. If curve changed, math recalculates it's state. Math is contained in separate class (BGCurveBaseMath). If you were using math and do not need it anymore, you'd better call math.Dispose() method ASAP. If you need math, we recommend you to use BGCcMath component, which gives you easy access to math operations and more. Read about BGCcMath under components section.

Math methods

1
Curve's distance ( math.GetDistance() )
2
Position by distance ( math.CalcPositionByDistance(float distance) or math.CalcPositionByDistanceRatio(float distanceRatio) [Range (0,1)])
3
Tangent by distance ( (math.CalcTangentByDistance(float distance) and math.CalcTangentByDistanceRatio(float distanceRatio) [Range (0,1)])
4
Position and distance(optionally) for a point, which is on the curve and closest to some other point (math.CalcPositionByClosestPoint(Vector3 point))

Understanding Components

Components are the standard Unity MonoBehaviour subclasses, which gives you access to some additional curve's related functions at Play mode without any scripting. We recommend to use them unless they do not cover your needs. For example, if you need math, do not create BGCurveBaseMath yourself, simply attach BGCcMath component and reference it instead of BGCurve.

Understanding Events

Curve.Changed event is fired once per frame in Update (or LateUpdate (configurable)) curve's method if curve is changed. Math uses this event to update it's cached data. if you need to update some data after math is changed, use math.Changed event instead of curve.Changed.
