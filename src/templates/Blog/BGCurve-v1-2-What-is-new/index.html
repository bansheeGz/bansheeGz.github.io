#set title=BG Curve v1.2: What is new? #set date=11/2016
  1. Custom fields for points
  2. Animation support
  3. Snapping (to terrain etc.)
  4. New components (Scale and Triangulate)
  5. Components (ChangeCursor and Rotate) are upgraded to support custom points fields
  6. New approximation type (Adaptive)
  7. Base math switched to the fastest algorithm available for approximation (~25% faster)
  8. Points store options (inlined, component, gameobjects)
  9. Vertex snapping (V key) for points configurable handles (in Scene View)
