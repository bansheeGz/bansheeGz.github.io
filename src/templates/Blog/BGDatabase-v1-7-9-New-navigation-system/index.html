Back/forward navigation setting

"Ignore navigation history if current entity is not set" parameter allows to ignore switching between different tables without selecting a row. When the row is selected it's highlighted with different color. If this row is not selected- then selecting a table will be ignored and will not be included into navigation history.

What if I do not like new navigation?

Navigation system can be disabled
