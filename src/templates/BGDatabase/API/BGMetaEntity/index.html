#set page=BGMetaEntity API

BGMetaEntity.cs -is a database table, containing 1-n fields and 0-n rows.

Main API

Method/Property Returns Description
this[int index] BGEntity Get table row by it's index 
//get first row
var entity = meta[0];
this[BGId id] BGEntity Get table row by it's id 
//get row by it's id
var entity = meta[new BGId("HThOdRCxlEGU1aefSG+Nbw")];
ForEachEntity(Action<BGEntity> action, Predicate<BGEntity> filter) void Search entities by filter. For each found entity action will be called 
//print entities names, whose name starts with "My"
meta.ForEachEntity(entity => Debug.Log(entity.Name), entity => entity.Name.StartsWith("My"));
CountEntities int The number of rows 
//print the number of rows
Debug.Log(meta.CountEntities);
NewEntity BGEntity Create new row 
//print the number of rows
var newEntity = meta.NewEntity();

Additional API

Method/Property Returns Description
Id BGId table id 
//get table id
var id = meta.Id;
Name string table name 
//get table name
var name = meta.Name;
ForEachField(Action<BGField> action) void Execute action for each field 
//print all field names
meta.ForEachField(field => Debug.Log(field.Name));
ForEachField(Action<BGField> action, Predicate<BGField> filter) void Execute action for each field, complying to the filter 
//print all field names, whose value type is int
meta.ForEachField(field => Debug.Log(field.Name),
                field => field.ValueType==typeof(int));
