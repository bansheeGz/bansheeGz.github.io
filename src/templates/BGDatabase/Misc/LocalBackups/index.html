#set page=Misc #set subPage=Local backups

Description

We provide local backups for database files, which are created automatically. Backups can be accessed from Setting->Local backups page.

There are 2 types of backups

  1. Instant backups - created immediately on database saving
  2. Daily backups - created once per day

Warning

Local backups are not 100% robust way to avoid any risk of data loss, because they can not help in case of your hard drive failure. That's why we highly recommend to regularly backup or commit database file into your Version Control system to avoid possible data loss risks.
