Description

The guide about importing and exporting data to CSV files

Current implementation of CSV files synchronization is very simplified compared to Excel/Google sheets synchronization, CSV files can only contain database data, "Update Ids on import" feature is not supported and many additional settings are not supported. During data exporting several CSV files are created (one per table)

Setup

  1. Read this guide to learn how to create a data source and a job and export/import data to CSV files
  2. Select a folder for CSV files on data source page. Export creates a separate CSV file for each table.
