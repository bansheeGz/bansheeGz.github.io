LiveUpdate add-on example

Version:
1.0
Format:
Unity project zip archive
Dependencies:
BGDatabase

Description

These guides and example projects demonstrate LiveUpdate add-on usage.

Setup:

  1. Download zip archive, unzip it and open the project in Unity Editor (version >= 2019 LTS). Ignore error messages.
  2. Import BGDatabase asset (if you use Playmaker-based project, import Playmaker as well)
  3. Open BGDatabase window
  4. Run scene (Scenes\SampleScene.unity)
