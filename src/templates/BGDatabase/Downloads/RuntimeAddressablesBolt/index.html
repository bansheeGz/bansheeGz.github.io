Custom action for Bolt and Addressables

Version:
1.1
Format:
Unity package
Dependencies:
BGDatabase, Addressables, Bolt

Description

Custom action (BGAddressablesGetAsset under BansheeGz category) for Bolt to load Unity assets from Addressables and BGDatabase without any prerequisites (no preloading required). Loading is asynchronous and may take several frames to load an asset.

Setup:

  1. Unit requires a Unity Event to be called in coroutine mode. Make sure Unity event has "Coroutine" toggle on
  2. Add BGAddressablesGetAsset unit. Assign correct meta name, field name and pass correct entity identifier (Index, ID or name). The call may take several frames to load an asset, cause loading is asynchronous
