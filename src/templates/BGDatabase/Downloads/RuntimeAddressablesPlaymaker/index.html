Custom action for Playmaker and Addressables

Version:
1.0
Format:
Unity package
Dependencies:
BGDatabase, Addressables, Playmaker

Description

Custom action (BGAddressablesGetAsset under BansheeGz category) for Playmaker to load Unity assets from Addressables and BGDatabase without any prerequisites (no preloading required)

Setup:

  1. Create UnityEngine.Object variable and assign required type (for example GameObject for prefab field)
  2. Add new State and add BGAddressablesGetAsset action. Assign required table, field and entity. Use variable, you created at step #1 as Result
  3. Use FINISHED event to move to the next state
  4. Use variable to access the asset. You may need to use "Convert" option to convert it to required type (for example, use GameObject prefab field to instantiate the prefab with standard "Create object" Playmaker action)

Available actions:

  1. BGAddressablesGetAsset - load asset from addressables system
  2. BGAddressablesReleaseAsset- release asset loaded from addressables
