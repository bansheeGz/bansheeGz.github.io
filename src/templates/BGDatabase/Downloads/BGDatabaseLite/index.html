Description

Basic free version of the package with following features:
  1. Creating and editing database (except addons)
  2. Creating and editing of delta files for Modding addon

End-user license agreement

  1. "BGDatabase Lite" package is free for non-commercial use only
  2. If you own BGDatabase package you can include "BGDatabase Lite" package to your game SDK to support modding
