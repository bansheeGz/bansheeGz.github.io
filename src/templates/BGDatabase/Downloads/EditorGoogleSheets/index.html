BGDatabase Google Sheets plugin

Version:
1.0
Format:
Unity package
Dependencies:
BGDatabase (v. >= 1.6)

Description

This plugin adds support for export/import to Google Sheets in Unity Editor.

Newtonsoft Dlls conflict

Some people already has Newtonsoft.Json.dll (JSON.NET) in their projects, and it conflicts with Newtonsoft.Json.dll, bundled with GoogleSheets API dlls. The following error is printed to the console in this case: "PrecompiledAssemblyException: Multiple precompiled assemblies with the same name Newtonsoft.Json.dll included or the current platform. Only one assembly with the same name is allowed per platform."

Here is how to resolve this issue:

  1. Select Newtonsoft.Json.dll, shipped with our package in the Project window, uncheck all platform toggles and hit "Apply" button
  2. Select all Google Sheets API DLLs, uncheck "Validate references" and hit "Apply" button
