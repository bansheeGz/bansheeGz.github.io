BGDatabase Google Sheets plugin

Version:
1.1
Format:
Unity package
Dependencies:
BGDatabase (v. >= 1.6.2)

Description

This plugin adds support for export/import to Google Sheets in Unity Editor.

Newtonsoft Dlls conflict

Some people already have Newtonsoft.Json.dll (JSON.NET) in their projects, and it conflicts with Newtonsoft.Json.dll, bundled with GoogleSheets API dlls. The following error is printed to the console in this case: PrecompiledAssemblyException: Multiple precompiled assemblies with the same name Newtonsoft.Json.dll included or the current platform. Only one assembly with the same name is allowed per platform.

Here is how to resolve this conflict:

  1. Select Newtonsoft.Json.dll, shipped with our package in the Project window, uncheck all platform toggles and hit "Apply" button
  2. Select all Google Sheets API DLLs, uncheck "Validate references" and hit "Apply" button

Using source code

We provide source code for our plugins, so you can use source code instead of built DLL file or build your own DLL using source code.

Guide for using source code:

  1. While importing plugin package, check the packages with source code only
  2. Download required library- Google APIs (min version 1.41.1) here and add library DLLs to your project to the Editor assembly (under Editor folder) (the following DLLs are required- Google.Apis.dll, Google.Apis.Auth.dll, Google.Apis.Auth.PlatformServices.dll, Google.Apis.Core.dll, Google.Apis.Sheets.v4.dll, Newtonsoft.Json.dll )
  3. Import source code packages
