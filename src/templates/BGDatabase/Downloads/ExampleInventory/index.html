Inventory system

Version:
1.0
Format:
Unity project zip archive

Description

Inventory system, built with BGDatabase (level=advanced). Approximate DB schema

Features

  1. Character creating with class and attributes
  2. Inventory
  3. Pick up items from containers
  4. Equip items
  5. Trading with traders
  6. Saving and loading

Setup:

  1. Download zip archive, unzip it and open the project in Unity Editor (version >= 2018 LTS)
  2. Import BGDatabase asset
  3. Run example scene (BGDatabaseInventoryExample/Scene/Main.unity)
