Features

  1. Fast, In Memory, pure C#, no SQL database fully integrated into Unity 3D
  2. Selective import/export from/to Excel and Google Sheets
  3. Selective Save/Load during game session
  4. Lazy loading
  5. Powerful code generation
  6. Playmaker support
  7. Hook up any GameObject to table row by adding a component
  8. Over 40 field types supported, including relations and nested table.
  9. Ability to create your own field.
  10. Runtime changes saving.
  11. Live update from Google sheets
