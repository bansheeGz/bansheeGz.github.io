#set page=Rotate Object By Cursor [Component] #set version=v11 #set imageBig=images/GifCcRotateByCursor.gif #set description=Rotates the object to curve's tangent or custom rotation field, which are determined by Cursor position. It uses Quaternion.LookRotation method. It does not change the cursor itself, you need another component to change cursor position (like Cursor Change Linear)
