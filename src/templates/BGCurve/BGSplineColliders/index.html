Description

Colliders pack consists of 3 components, which generate colliders along Bezier spline.

BGCcCollider3DBox

Generate BoxCollider for 3D spline

BGCcCollider2DBox

Generate 2D BoxCollider for 2D spline

BGCcCollider2DEdge

Generate points for 2D EdgeCollider for 2D spline
