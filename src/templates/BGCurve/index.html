Quick Demo and Tutorial

Features

  • Single Game Object (no additional objects for points)
  • Base math for Bezier curves (get Total Distance, get Position by distance, get Tangent by distance, get Position and Distance by Closest point to a curve)
  • 2D mode
  • Components (no scripting required)
  • Rectangular selection for points
  • Context menu for points and selections in SceneView
  • New points has preview in Scene View before adding
  • A lot of Editor settings
  • Save/Load settings
