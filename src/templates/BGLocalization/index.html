Features

  1. Free copy of BG Database as it's built on top of BG Database (all BGDatabase features, like Excel/Google Sheets export and import, are available)
  2. Localize string/text or any Unity Asset (assets should be placed under Resources or in asset bundle)
  3. Target any component, any field or property (reflection is used)
  4. Machine Translation is available: 1) Amazon Translate- builtin, 2) Yandex Translate- builtin, 3) IBM Watson Translate (some restrictions apply), 4) Google Translate (via Google Sheets only)
