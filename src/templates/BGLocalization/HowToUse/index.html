Setup

  1. Import BGLocalization package
  2. Create your own database ( more info)
  3. Enable localization addon
  4. Go to Database->Localization and add your locales. Locale name- letters only, 8 chars maximum.
    Note, locales is just a regular table, so you can add your own fields, like native spelling, flag icons etc.
  5. Go to Configuration and create a table for your locale data (type of the table is Localization), chose the type of data to be stored.
  6. Create a normal table (row)
  7. Add a localized field to this table
  8. Go to Database->"Your created table", add some rows. You can edit localized values here (current locale will be used)
  9. Another option- is to chose "localization" table and edit values for all locales at once. You can use Auto-translation here
  10. Learn more about auto-translation here
  11. Once you data is ready, you can localize your Components (any field or property). Add BGLocalizationGo to your GameObject, chose one of the "localization" tables, assign an entity and chose target component and field/property

How to change locale

    var addon = BGRepo.I.Addons.Get<BGAddonLocalization>();
    addon.CurrentLocale = "en"; // replace en with your locale name
